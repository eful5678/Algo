# Implementation

## 1. Problem #1 - 시각

---

- 정수 N이 입력되면 00시 00분 00초부터 N시 59분 59초까지의 모든 시각 중에서 3이 하나라도 포함되는 모든 경우의 수를 구하는 프로그램
- Example
  - Input : 1
    - 00시 00분 03초
    - 00시 13분 30초
  - Input : 5
  - Output : 11475
- 조건
  - Input
    - 첫째 줄에 정수 N 입력(0 <= N <= 23)
  - Output
    - 00시 00분 00초부터 N시 59분 59초까지의 모든 시각 중에서 3이 하나라도 포함되는 모든 경우의 수 출력
