# Algorithm

## 검색 알고리즘

---

1. Linear Search

```
For i from to n-1
    If i'th element is 50
        Return true
Return false
```

2. Binary Search

```
If no item
    Return false
If middle item is 50
    Return true
Else if 50 < middle item
    Search left half
Else if 50 > middle item
    Search righ half
```
